# Jarurat Care Foundation – Mini Healthcare Support Web App

## Project Description

This project is a concept-level healthcare support web application created as an internship assignment for the Full Stack Developer (AI-enabled) Internship at Jarurat Care Foundation.

The aim of this project is to provide an easy and user-friendly way for patients, volunteers, and visitors to interact with the NGO through a simple web interface. It is designed as a frontend prototype to demonstrate responsive UI design, form handling, and basic automation concepts for a healthcare support system.

## Features

- Responsive design
- Patient Support Form
- Volunteer Registration Form
- Contact Form
- FAQ Chatbot
- Auto-generated Request ID
- Success Modal
- Dark/Light Mode
- Smooth Scrolling
- Animated Statistics
- Mobile Friendly UI

## AI / Automation Idea

This project includes a rule-based FAQ chatbot and automatic request confirmation with generated request IDs as the AI/automation concept for the assignment. These features demonstrate how simple automation can improve user interaction and make support requests feel more organized and professional.
