# Jarurat Care Foundation — Frontend Prototype

This project is a concept prototype developed for the Jarurat Care Foundation Full Stack Developer Internship Assignment.

It is a single-page frontend-only demo created with HTML, CSS, and JavaScript. The forms and chatbot are rule-based/demo-only and do not submit data to a backend.

Run locally (recommended using VS Code Live Server):

1. Open the folder in VS Code.
2. Install the "Live Server" extension and click "Go Live" while `index.html` is open.

Or use Python (if available):

```powershell
py -m http.server 8000
# then open http://127.0.0.1:8000
```

Notes:
- All demo values and testimonials are clearly marked with "(Demo)" or labelled as demo values.
- Do not treat this as production-ready software — it is a design-and-interaction prototype for evaluation purposes only.
